This site exists for three main reasons. The biggest is that there are many teaching focused institutions without the space or resources for experimentalists to engage in meaningful computer-based research. By opening the source for this website, using python and javascript, and offering researchers the opportunity to collaborate with me using this platform, I hope to make it easier for social-science researchers at all institutions to engage in meaningful computer-based experimental, correlational, and descriptive research, regadless of the amenities provided at their individual institutions.

Second, given the relatively recent and ongoing issues related to reproducibility and replication, I wanted to design a platform where the materials and results are open to everyone. Researchers concerned about the validity of data that comes from online sources, could easily conduct their own replication efforts in the lab using much of the same code provided at the Github repository for this project—if you happen to be a expert in javascript or python and are interested in this project, please feel free to fork the project and submit a pull request with any helpful changes.

Finally, although there are a number of websites that offer students the opportunity to participate in many classical psychological experiments. The underlying methods haven't been made entirely clear, making it difficult to accurately describe the methods, should anyone investigate new hypotheses with their ever growing data sets. By providing access both to the specific materials and results, investigators are free to explore the data in search of evidence (dis)confirming novel hypotheses.

Over time, the number of experiments offered on this website should grow to include a number of classical demonstrations. Contrasted with the "Many Labs Replication Project" and given the online nature of this project, this site can be viewed as a "No Labs Replication Project."
